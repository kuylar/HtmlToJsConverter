HTML to JavaScript Converter

This is a quick tool to convert your HTML snippets to JavaScript functions, with a few extras.

Try it live!



Documentation

Inserting text from model

You can take values from the model and use them everywhere by using the {{ modelName }} syntax in the input

Example

Input 
<h5>Hello {{ name }}</h5>
Build 
buildComponent({ name: "world" });
Result 
<h5>Hello world</h5>

Lists/Arrays

Example

Input 
<h1>{{ title }}</h1>
<ul kjs-list="items">
  <li>{{ item }}</li>
</ul>
Build 
buildComponent({
  title: "Shopping List",
  items: [
  "Milk",
  "Eggs"
]});
Result 
<h1>Shopping List</h1>
<ul>
  <li>Milk</li>
  <li>Eggs</li>
</ul>

Lists will also be returned in the returned object inside lists, so you can add items by using result.lists["items" OR value of kjs-list].append({model})

Evaluating JavaScript

You can use the {! javascriptCode !} syntax to evaluate a JavaScript snippet inside the model

Example

Input 
<p>There are 
  {! model.items.length !}
  items in the list</p>
Build 
buildComponent({ items: [
  "Apple",
  5,
  {fruit: "Banana"}
]});
Result 
<p>There are 3 items in the list</p>

Most features are enabled with HTML attributes.

  • kjs-insertType="<text|html>" Specifies to the method of inserting the text in the element. text uses innerText and html uses innerHTML
  • kjs-parserName="<name>" Gives the element the name in the attribute, and returns it in the resulting object
